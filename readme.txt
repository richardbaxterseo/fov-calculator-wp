A professional field of view calculator for sim racing setups. Calculate the perfect FOV for your monitors.

== Description ==

The FOV Calculator for Sim Racing helps sim racers calculate the optimal field of view settings for their gaming setup. Simply add the calculator to any page or post using the [fov_calculator] shortcode.

**Features:**

* Support for single and triple screen setups
* Multiple screen ratio options (16:9, 21:9, 32:9, and more)
* Curved monitor support with adjustable radius
* Calculations for popular sim racing games including:
  - iRacing
  - Assetto Corsa & ACC
  - Project CARS series
  - rFactor series
  - F1 games
  - And many more!
* Beautiful, responsive design
* Server-side calculations for security
* No external dependencies

**Usage:**

Simply add `[fov_calculator]` to any page or post where you want the calculator to appear.

== Installation ==

1. Upload the `fov-calculator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the `[fov_calculator]` shortcode in any page or post

== Frequently Asked Questions ==

= How do I add the calculator to my site? =

Simply add the shortcode `[fov_calculator]` to any page or post where you want the calculator to appear.

= Can I customize the appearance? =

The calculator comes with a modern, responsive design that works well with most themes. Custom CSS can be added through your theme if needed.

= Is this calculator accurate? =

Yes! The calculations are based on proper trigonometry and have been tested against other popular FOV calculators.
